Sakura web

Sakura is a Discord bot built using Django, a Python web framework. It offers various functionalities such as automated moderation, customized commands, and Fun commnads. Its name is inspired by the Japanese word for cherry blossom, a symbol of new beginnings and growth.

Yae Sakura

Yae Sakura is a Discord bot built using the Command Line Interface (CLI) that offers admin and moderation commands, as well as fun commands for entertainment. Additionally, it has a feature that allows users to assign themselves roles by reacting to messages. Its name is inspired by the character Yae Sakura from the popular mobile game, Honkai Impact 3rd.

Custom Android Rom

Android custom ROMs are modified versions of the Android operating system that offer different features and customization options than the stock Android OS. Neutron and Wrath OS are two custom ROMs that have been built by me and have been released as official builds for the Redmi YSL device(Redmi Y2/S2). These ROMs may offer different user interfaces, improved performance, and unique features not found on the stock Android OS.

Zizuhu Kernel

Zizuhu is an Android kernel designed specifically for the Redmi Y2/ YSL devices. It has been integrated into both the Wrath and Neutron OS custom ROMs, providing users with an optimized experience. The kernel offers a boost in performance specifically for gaming, which can enhance the user's gameplay experience. The name "Zizuhu" is inspired by the sound of a buzzing bee, which represents speed and efficiency.
